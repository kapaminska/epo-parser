# EPO Parser

Offline desktop tool for Lasów Państwowych: converts e-Doręczenia EPO XML files to readable PDFs in the working directory, with a text summary (`epo-konwersja.txt`).

Product docs: `context/foundation/prd.md`, `context/foundation/tech-stack.md`, `context/foundation/roadmap.md`.
